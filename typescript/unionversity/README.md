# Unionversity (TypeScript)

Codecademy project focused on **union types**, **type guards**, and **type-safe searching/enrollment**.

## Concepts practiced

* Union types (`string | number`, literal unions)
* Type guards (`typeof`)
* Type inference
* Working with arrays (`filter`, `includes`)
* `tsconfig.json` basics

## Files

* `index.ts` — main program (search + enroll)
* `courses.ts` — course data
* `studyGroups.ts` — study group data
* `tsconfig.json` — TypeScript compiler settings
